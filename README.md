# Herbivore diet code

This repository contains the R code used to generate figures for the herbivore diet composition analyses.

## Files

- `Code 1 - Clustering.R`  
  Code used for diet composition clustering, heatmaps, and ordination plots.

- `Code 2 - Comparing compositions.R`  
  Code used to compare diet composition between sites/regions.

- `Code 3 - body size.R`  
  Code used for the diet diversity and body-size analyses.

- `Code 4 - Korona plots.R`  
  Code used to generate Krona/Korona-style supporting visualizations.

- `Supporting data - RRA.xlsx`  
  Supporting relative read abundance data used by the scripts.

## Notes

The scripts are provided to show the code used to generate the figures and supporting visualizations for the analysis.
