# Herbivore diet code

This repository contains the R code used to generate the figures and supporting visualisations for the herbivore diet composition analyses.

## Files

- `Supporting data - RRA.xlsx`  
  Input relative read abundance data used by the R scripts.

- `Code 1 - Clustering.R`  
  Code used for diet composition clustering, heatmaps, and ordination plots.

- `Code 2 - Comparing compositions.R`  
  Code used to compare diet composition between datasets/sites/regions.

- `Code 3 - body size.R`  
  Code used for the diet diversity and body-size analyses.

- `Code 4 - Korona plots.R`  
  Code used to generate Krona/Korona-style visualisations.

## Notes

The Excel file is the input data file used by the scripts. The scripts are provided to show the code trail used to generate the figures and supporting visualisations.
